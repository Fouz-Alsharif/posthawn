# HAWN Static Interface

This project has been converted to a plain HTML, CSS, JavaScript, and JSON interface.

## Open with Live Server

1. Open this folder in VS Code.
2. Right-click `index.html`.
3. Choose `Open with Live Server`.

The first aid recommendations are loaded from `first-aid-guidance.json`. The image analysis result is still mocked in `script.js` until a real YOLO/API output is connected.
